## Overview

Troubleshooting network services focuses on resolving issues with IP addressing, DHCP, DNS, routing, switching loops, firewalls, and VLAN configurations.

## Topics

- [[001 Troubleshooting Network Services|Troubleshooting Network Services]] — Overview of network service troubleshooting
- [[002 Duplicate Addresses|Duplicate Addresses]] — Detecting and resolving IP address conflicts
- [[003 DHCP Issues|DHCP Issues]] — DHCP server failures, scope exhaustion, and relay problems
- [[004 Routing Issues|Routing Issues]] — Missing routes, misconfigurations, and black holes
- [[005 Switching and Routing Loops|Switching and Routing Loops]] — STP failures, routing loops, and TTL issues
- [[006 Firewall Issues|Firewall Issues]] — Overly restrictive rules and asymmetric routing
- [[007 IP Configuration Issues|IP Configuration Issues]] — Wrong subnet mask, gateway, or DNS settings
- [[008 VLAN Issues|VLAN Issues]] — Trunk mismatches, allowed VLANs, and native VLAN
- [[009 DNS and NTP Issues|DNS and NTP Issues]] — Resolution failures, time sync, and DNSSEC problems

## Key Concepts

- DHCP troubleshooting (DORA failures, scope exhaustion)
- Routing loop detection and prevention
- Firewall rule analysis and logging
- VLAN trunk and tagging verification
