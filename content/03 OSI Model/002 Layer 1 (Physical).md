## 📖 Core Concept
- The **Physical Layer** is where the transmission of bits across the network occurs.
- It defines the physical and electrical network characteristics.
- Binary bits (0's and 1's) are the basic building block of this layer.

## 🛠️ Hardware & Devices
- **Cables:** Fiber optic, Ethernet (Cat cables), Coaxial.
- **Connectors:** RJ45 (for Ethernet).
- **Network Devices:** Hubs, Access Points, Media Converters.
- **Wireless Tech:** NFC, Wi-Fi, Radio.

## 🔑 Key Terminology
- **Transition Modulation:** If the signal changes during the clock cycle, a `1` is represented. Otherwise, a `0` is represented.
- **Asynchronous:** Uses start and stop bits to indicate where transmission occurs from the sender to the receiver.
- **Synchronous:** Uses a shared reference clock to coordinate the transmission between the sender and receiver.
- **Baseband:** Uses *all* of the bandwidth allocated to that connection (uses a reference clock).
- **Broadband:** Divides bandwidth into separate channels.
- **Frequency-Division Multiplexing (FDM):** Dividing the medium into channels based on frequencies. Each session is transmitted over a different channel (similar to how broadband works). Multiplexing allows multiple users to share the same bandwidth.
