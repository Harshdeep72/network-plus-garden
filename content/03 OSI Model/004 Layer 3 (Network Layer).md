## 🧠 Main Takeaways
- there are 2 logical addressing we use that is ipv4 and ipv6 while we use these 2 because these are most common there are more then these 2 .
- how do we forward this data? there are three ways we can do it :
	- ==packet switching== ( most commonly used also know as routing) in this data is being divided into packets and then forwarded. (more research needed)
	- Circuit switching
	- ==Message switching Data is divided into messages which may be stored and forwarded==
- Route Discovery and Selection : we configure this manually as a static route or dynamically though routing protocol ( RIP , OSPF , EIGRP )
- routing protocol help us decide how data is going to flow across the network and how routers are going to communicate
- flow control ( Augment of layer 2 )
- packet reordering ( Augment of layer 2 ) it allows us to get big chunk of data and cut it up in small pieces and then send it to all different directions to get to their final destination now the packet reordering ensures that all data reaches the receiver correctly at the end of destination.
- ICMP (internet Control Message Protocol ) : sends error messages and operational information to an IP destination.
