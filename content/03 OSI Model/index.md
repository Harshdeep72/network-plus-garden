## Overview

The OSI model is a conceptual framework that standardizes networking functions into seven distinct layers, from physical transmission to application services.

## Topics

- [[001 OSI Model|OSI Model]] — Overview of the seven-layer OSI model
- [[002 Layer 1 (Physical)|Layer 1 (Physical)]] — Physical transmission of raw bits over media
- [[003 Layer 2 (Data Link Layer)|Layer 2 (Data Link Layer)]] — Framing, MAC addressing, and error detection
- [[004 Layer 3 (Network Layer)|Layer 3 (Network Layer)]] — Logical addressing, routing, and packet forwarding
- [[005 Layer 4 (Transport Layer)|Layer 4 (Transport Layer)]] — Reliable data delivery and flow control
- [[006 Layer 5 (Session Layer)|Layer 5 (Session Layer)]] — Session management and dialog control
- [[007 Layer 6 (Presentation Layer)|Layer 6 (Presentation Layer)]] — Data formatting, encryption, and compression
- [[008 Layer 7 (Application Layer)|Layer 7 (Application Layer)]] — User-facing network services and protocols
- [[009 Encapsulation and Decapsulation|Encapsulation and Decapsulation]] — How data moves through the OSI layers
- [[010 Understanding the OSI Model|Understanding the OSI Model]] — Practical understanding of the OSI model

## Key Concepts

- Seven-layer abstraction model
- Encapsulation and decapsulation
- Protocol data units (PDUs) at each layer
- Layer interaction and data flow
